## App environment variables

- `API_MOCKING`: set to `enabled` to enable
- `API_TOKEN`: graphcms api token
- `API_URL`: graphcms api url

## Worflow environment variables

- `GITHUB_TOKEN`
- `PERCY_TOKEN`
- `VERCEL_PROJECT_ID`
- `VERCEL_TEAM_ID`
- `VERCEL_TOKEN`
