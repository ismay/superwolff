# Superwolff

> Portfolio website

## Build environment variables

- `ANALYZE_BUNDLE`: set to `enabled` to enable
- `API_MOCKING`: set to `enabled` to enable
- `API_TOKEN`: graphcms api token
- `API_URL`: graphcms api url
- `CLEAN_MOCKS`: set to `enabled` to enable
